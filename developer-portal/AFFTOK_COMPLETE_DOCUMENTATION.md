# 🎯 نظرة عامة

## ما هو AffTok؟

AffTok هو منصة متكاملة لتتبع التسويق بالعمولة (Affiliate Marketing) مصممة للشركات والمعلنين والناشرين. توفر المنصة حلاً شاملاً لإدارة الحملات الإعلانية، تتبع النقرات والتحويلات، وتحليل الأداء في الوقت الفعلي.

## المبدأ الأساسي

```
┌─────────────────────────────────────────────────────────────────┐
│                   🎯 دور المنصة الحصري                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ✅ ما نفعله:                                                  │
│   ├── تتبع النقرات والتحويلات                                   │
│   ├── عرض العروض للمروجين                                       │
│   ├── إحصائيات وتقارير                                          │
│   └── أخذ نسبة تقنية 10%                                        │
│                                                                 │
│   ❌ ما لا نفعله:                                               │
│   ├── وساطة مالية                                               │
│   ├── معالجة مدفوعات                                            │
│   ├── حكم في النزاعات                                           │
│   └── التحقق من صحة المبيعات                                    │
│                                                                 │
│   💡 المنصة = تتبع + عرض + عمولة تقنية                          │
│      المنصة ≠ وسيط مالي / قاضي / محاسب                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

# 🔧 Backend API (Go)

## نظرة عامة

Backend مبني بـ Go مع إطار Gin، يوفر REST API عالي الأداء.

## الهندسة المعمارية

```
┌─────────────────────────────────────────────────────────────────┐
│                      Backend Architecture                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Request → Router → Middleware → Handler → Service → Database  │
│                                                                 │
│  ┌─────────┐  ┌─────────────┐  ┌─────────┐  ┌──────────────┐  │
│  │ Router  │→ │ Middleware  │→ │ Handler │→ │   Service    │  │
│  │ (Gin)   │  │ (Auth/CORS) │  │ (HTTP)  │  │   (Logic)    │  │
│  └─────────┘  └─────────────┘  └─────────┘  └──────┬───────┘  │
│                                                     │          │
│                                              ┌──────┴──────┐   │
│                                              │  Database   │   │
│                                              │ (PostgreSQL)│   │
│                                              └─────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## المسارات الرئيسية (Routes)

### التتبع (Tracking)
```
GET  /c/:id                    # تتبع النقرة وإعادة التوجيه
GET  /r/:code                  # صفحة المروج
GET  /@:username               # صفحة المروج بالاسم
POST /api/postback             # استقبال التحويلات
```

### المستخدمين (Users)
```
POST /api/auth/register        # تسجيل جديد
POST /api/auth/login           # تسجيل دخول
GET  /api/auth/me              # بيانات المستخدم الحالي
PUT  /api/user/profile         # تحديث الملف الشخصي
```

### العروض (Offers)
```
GET  /api/offers               # قائمة العروض
GET  /api/offers/:id           # تفاصيل عرض
POST /api/offers/:id/join      # الانضمام لعرض
GET  /api/offers/my            # عروضي
```

### الفرق (Teams)
```
GET  /api/teams                # قائمة الفرق
POST /api/teams                # إنشاء فريق
GET  /api/invite/:code         # صفحة دعوة الفريق
```

### المسابقات (Contests)
```
GET  /api/contests             # قائمة المسابقات
GET  /api/contests/active      # المسابقات النشطة
```

## التحقق من التوقيع

```javascript
const crypto = require('crypto');

function verifyWebhook(payload, signature, secret) {
  const expectedSignature = crypto
    .createHmac('sha256', secret)
    .update(payload)
    .digest('hex');
  
  return signature === expectedSignature;
}
```

CREATE TABLE clicks (
